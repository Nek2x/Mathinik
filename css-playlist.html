Confidence Intervals

It is a range of values used to estimate an unknown population parameter, such as a mean or proportion. It provides not just a single number but a range where the true value is likely to fall. The interval comes with a confidence level (like 95%) that tells us how sure we are that the true value lies within that range. Confidence intervals help us express uncertainty in estimates and are commonly used in surveys, experiments and data analysis.
